# FIDO Alliance WebAuthn demo

Slides: http://slides.com/herrjemand/jan-2018-fido-seminar-webauthn-tutorial#/

### Requirements

- Firefox Nightly
- Nodejs + NPM
- Text editor

### Run

- `git clone https://github.com/fido-alliance/webauthn-demo/`
- `cd webauthn-tutorial`
- `npm install`
- `node app`
